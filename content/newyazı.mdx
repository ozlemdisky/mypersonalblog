hellööö arkadaşlar. web tasarım front-end alanında sıklıkla karşılaşıp kullanılan iki programlama dilinden ufacık bahsetmek istiyorum.

CSS tek başına gayet güzel işler çıkarabiliyor elbet. Fakat daha büyük karmaşık stil sayfalarında bazı noktalarda yetersiz kalıyor. İşte tam bu noktada devreye SASS giriyor.

2006 yılında Hampton Catlin arkadaşımız tarafından çıkarılmış ve hala geliştirilmesi devam etmekte olan SASS, CSS ön işlemcisidir. henüz css’de bulunmayan , css yazmamıza yardımcı olacak mixin, döngüler, miras, nesting, gibi bazı özelliklere sahiptir.

SCSS ise mantık ve kullanım olarak SASS ile aynıdır. SASS, Ruby programa dili mantığıyla geliştirildiği için CSS’te yer alan ; ve { } işaretleri kullanılmamaktadır. Bu durum kullanıcılar için zorluk çıkarmaktaydı. Mayıs 2010'da SASS 3.0 sürümle beraber SASS ve CSS arasındaki bu eksiği kapatmak adına CSS için SCSS adlı yepyeni bir sözdizimi sunuldu. CSS ile tam uyumlu olması bu dizimin en büyük artısıdır.

SASS ve SCSS dosyaları compile edilerek CSS dosyalarına dönüştürülerek kullanılmaktadır.
